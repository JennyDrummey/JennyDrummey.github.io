UI Terms for the SKP

Use the following guidelines when referencing items on the User Interface of Syniti.

Term Location/Description Comment
Add Goal page

Add Goal page

In Strategy, pages where you add Goals and Initiatives look like their page name is actually the Program name. These pages open in edit mode if you click the "Add Goal" or "Add Initiative" Button at the top of the strategy asset list.

Add Initiative page

Add Initiative page

In Strategy, pages where you add Goals and Initiatives look like their page name is actually the Program name. These pages open in edit mode if you click the "Add Goal" or "Add Initiative" Button at the top of the strategy asset list.

Admin menu

Displays above About Me, Accounts and Sites tabs

 
card

Cards are on the Search Results page.

Don't capitalize. For example:

The search results card
list box

Clickable element that expands and displays a list that users can make selection(s) from.

Incorrect nomenclature for this element includes:

  • drop-down
  • drop-down menu
  • drop-down list
  • drop down

Save

Whenever in edit mode, the Save button displays for users to save their updates.

Use the following standard:

Click Save.

Do not use the following:

Click the Save button.
