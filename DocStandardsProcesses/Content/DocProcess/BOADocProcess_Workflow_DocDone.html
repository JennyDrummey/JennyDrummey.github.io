Workflow - Doc Done

Build help in the dev/prod environemt in jenkins

If the updates to help are for a version that has been released, build the help in dev and then in prod.

If the updates to help are for a version that has not been released, you can build in dev as needed.

Refer to Update the Help on Dev and Prod sites for steps to push the changes to the dev and prod sites as needed.

Change status
