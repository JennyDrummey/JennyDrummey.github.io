Search and Replace Code within Tags in Flare

The Search and Replace function for Flare has been upgraded to allow us to update code that's in tags. For Flare to make the updates, the attributes within the tags must be updated one at a time.

To update attributes within tagged code in Flare:

  1. Click the Home tab on the Flare toolbar.
  2. Click the Find and Replace icon; the Find and Replace in Files window displays on the right.
  3. Click the Find Elements tab.
  4. Under Find:, select <> from the drop down list.
  5. Enter the name of the tag in the Tag combo box and select the correct tag from the drop down list.

  6. Click the Add Row button to indicate one attribute that defines the tag.

    NOTE: Only one attribute may be updated at a time. You may use multiple attributes to define a tag, but all updates you wish to make to attributes must be executed separately.

  7. Under Replace/Action:, click <> to select the action you want to perform with the Search and Replace. The options are:

    • Set Tag

    • Remove Tag

    • Unbind Tag

    • Wrap Tag

    • Add Class

    • Remove Class

    • Replace Class

    • Remove Style ID

    • Set Style ID

    • Remove Inline Formatting

    • Set Inline Formatting

    • Remove Attribute

    • Set Attribute

  8. Under Preview, review the syntax generated by your selections to ensure accuracy of the operation.

  9. Select an action to perform. The options are:

    • Find Next

    • Find Previous

    • Skip File

    • Find All

    • Replace

    • Replace All

  10. Click the Save All icon to save changes made to your files.
