Add Duplicate Links for Solex Versions

When a Solex version of the product is created (roughly twice a year), links between the SST and the Installation Manuals must be updated. All links between the Installation Guide project and the SST help must have a Full version and a Solex version of that link so that the user is directed to the correct documentation.

To add duplicate links in the SST help as a result of a solex release:

  1. Locate external links to the Installation Manual by running the External Links report in Flare.

  2. Apply the dspConduct CT to the Full version link

  3. Duplicate the link text, but apply the ISA CT to this Solex version link.

  4. Add the hyperlink to the correct documentation in the Installation manual to each link

NOTE: Watch for extra spaces as you add duplicate links.

To add duplicate links n the Installation Manuals project as a result of a solex release:

  1. Locate external links to the Installation Manual by running the External Links report in Flare.

  2. Apply the BOA CT to the Full version link.

  3. Apply the Install Solex CT to the Solex version link.

  4. Add the hyperlink to the correct documentation in the SST to each link.
