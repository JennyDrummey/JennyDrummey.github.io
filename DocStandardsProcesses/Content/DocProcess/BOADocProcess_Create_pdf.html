Convert Word to PDF

Typically, files of documents are distributed as secured PDF files â€“ not Word files.

To convert a Word document to PDF:

  1. Open the Word document.

  2. Click File > Save As> Adobe PDF

    Or

    Select Create PDF from the Acrobat tab.

  3. Select the file path.

  4. Enter a name in the File Name field.

  5. Click the Options button.

  6. Check the Convert Word Headings to Bookmarks check box.

  7. Uncheck the Convert Word Styles to Bookmarks check box.

  8. Click Save.

Secure PDF Files

Once the file has been created as a PDF, it has to be secured according to Syniti policy. Refer to [Security Settings for Adobe][BOADocProcess_DevManConfigGuides.html#securitysettings] for instructions on creating security policy.

To secure a PDF file:

  1. Open the PDF document.

  2. Select File > Properties.

  3. Click the Initial View tab.

  4. Select Bookmarks Panel and Page from the Navigation tab list box.

  5. Click OK.

  6. Select Tools.

  7. Expand Protection.

  8. Select Encrypt.

  9. Select CranSoft.

  10. Click Save.

Security Settings for Adobe

To protect the integrity of the documentation ,Syniti requires that all PDF files to be distributed to be protected by a standard Adobe security policy.

A security police is set up once for an installation of Acrobat.

To create a security policy in Adobe:

  1. Launch Adobe.

  2. Select View > Tools > Protection > Encrypt.

  3. Click Manage Security Policies.

  4. Click New.

  5. Click Use Passwords.

  6. Click Next.

  7. Click Save these settings as policy.

  8. Type a name (i.e.Â CranSoft) into Policy Name field.

  9. Click Next.

  10. Click Restrict editing and printing the document. A password will be required in order to change these persmission settings.

  11. Select High Resolution from Printing allowed list box.

  12. Select None for Changes allowed list box.

  13. Enter triangle in the Change Permissions Password field.

  14. Click OK; a Confirm Permissions Password window displays.

  15. Enter triangle in the Permissions Password field.

  16. Click OK.

  17. Confirm the information is correct.

  18. Click Finish.
