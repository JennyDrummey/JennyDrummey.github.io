Documentation Standards Overview

The Syniti Documentation team is responsible for producing the following:

  • Online help in Flare for SST
  • Online help, static guidance, API documentation and inline help in Flare for SKT
  • Installation Manuals and Configuration Manuals for each SST release in Flare
  • Developer Manuals and Technical Configuration Guides in the form of KB articles and PDF files
  • Data Insiders posts based on cConnect content
  • Knowledge Base articles in Zendesk. Support enters tickets on the DocTeam ToDos board to requests these articles, which may describe a workaround, or provide an overview of a process or feature (Example: DT-63 and DT-150).
  • Documentation Standards and Processes(this system)

Refer to Documentation Process for information about how the Documentation Team updates the online help.
